# Change: Add Frontend Application

## Why
A React SPA provides a user-friendly interface for managing funds, viewing cap tables, and executing transfers. Using a generated API client ensures type safety and eliminates manual HTTP handling.

## What Changes
- Create React + TypeScript + Vite project
- Generate API client from OpenAPI spec
- Implement dashboard, fund detail, and transfer UI
- Add TanStack Query for data fetching
- Style with Tailwind CSS
